# Personal Portfolio Website

Student Name: Muhammad Hasaan

## Objective

The objective of this project is to design and develop a simple, static personal portfolio website using HTML. The portfolio showcases the student's background, technical skills, education, and contact information in a structured, easy-to-navigate format.

## Technologies Used

- HTML5

## Features Implemented

- Header Section : Displays the name and professional title.
- Navigation Bar : Quick links to jump to About Me, Skills, Education, and Contact sections.
- Hero Section : A brief introductory welcome message.
- About Me Section : Summary of academic background, technical experience, and interests.
- Skills Section : List of technical skills (JavaScript, React.js, Node.js, MongoDB, Express.js, SQL/MySQL, n8n).
- Education Section : Details of degree program, university, and graduation year.
- Contact Section: Email, phone number, and location for reachability.
- Footer: Copyright notice.

## Project Description
This project is a static personal portfolio website built with HTML. It presents the student's professional background, technical skills, education, hobbies, and social/contact links in a clean, single-page, section-based layout with in-page navigation.

Project Description

This project is a static, multi-page business website built with HTML for a fictional web development company, "RavTech Limited." It consists of four interlinked pages Home, About, Services, and Contact sharing a consistent header, navigation, and footer, and presents the company's offerings, team, and contact channels.

Pages & Features Implemented


- Home (index.html) : Hero introduction, a "Who We Are" company summary, and a Featured Services preview with a link through to the full Services page.
- About (about.html) : Company overview, mission, and vision statements, plus a "Meet Our Team" section profiling four team members and their roles.
- Services (services.html) : Detailed breakdown of five core service offerings (Website Design & Development, Web Application Development, UI/UX Design, Maintenance & Support, and Automation Solutions), each with a description and feature list.
- Contact (contact.html) : Business address, email, and phone details via a semantic <address> element, plus a working contact form (name, email, subject, message) for direct inquiries.
- Shared Header & Navigation : Consistent site title and navigation menu linking all four pages, present on every page.
- Shared Footer : Copyright notice, contact email, and social media links (Facebook, Twitter, LinkedIn, Instagram) repeated across all pages.


## How to Run the Project

1. Download or clone the project folder to your local machine.
2. Ensure "index.html" (and any accompanying image files) are in the same folder.
3. Open "index.html" directly in any web browser (double-click the file, or right-click → Open With → your preferred browser).
4. No server, installation, or build steps are required, as this is a static HTML page.
